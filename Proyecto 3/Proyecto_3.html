<a>

Descripción: La etiqueta <a> define un hipervínculo, el cual es utilizado para enlazar una página a otra.

Ejemplo:

<a href="https://www.ejemplo.com">Enlace a Ejemplo</a>

<abbr>

Descripción: La etiqueta <abbr> define una abreviatura o acrónimo. Usualmente se usa con el atributo title para proporcionar una expansión o descripción.

Ejemplo:

<abbr title="World Health Organization">WHO</abbr>

<address>

Descripción: La etiqueta <address> proporciona información de contacto sobre el autor o propietario del documento o sección.

Ejemplo:

<address>123 Main St, Hometown, USA</address>

<area>

Descripción: La etiqueta <area> define un área dentro de un mapa de imagen.

Ejemplo:

<map name="map1">
    <area shape="rect" coords="34,44,270,350" href="https://www.ejemplo.com">
</map>

<article>

Descripción: La etiqueta <article> define un contenido independiente dentro de un documento, como un artículo en un blog o una noticia.

Ejemplo:

<article>
    <h2>Título del Artículo</h2>
    <p>Contenido del artículo...</p>
</article>

<aside>

Descripción: La etiqueta <aside> define contenido aparte del contenido principal del documento. Usualmente se usa para contenido relacionado como barras laterales.

Ejemplo:

<aside>
    <p>Contenido adicional relacionado...</p>
</aside>

<audio>

Descripción: La etiqueta <audio> se usa para incrustar contenido de audio en el documento.

Ejemplo:

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Tu navegador no soporta el elemento de audio.
</audio>

<b>

Descripción: La etiqueta <b> define texto en negrita. A diferencia de <strong>, no tiene un significado semántico especial.

Ejemplo:

<b>Texto en negrita</b>

<base>

Descripción: La etiqueta <base> especifica una URL base para todos los enlaces relativos en un documento.

Ejemplo:

<base href="https://www.ejemplo.com">

<bdi>

Descripción: La etiqueta <bdi> aísla un fragmento de texto para evitar que el texto en el interior afecte la dirección de texto en el exterior.

Ejemplo:

<bdi>Texto a aislar</bdi>

<bdo>

Descripción: La etiqueta <bdo> se usa para sobrescribir la dirección de texto predeterminada en un documento.

Ejemplo:

<bdo dir="rtl">Texto en dirección de derecha a izquierda</bdo>

<blockquote>

Descripción: La etiqueta <blockquote> define una cita larga o importante, usualmente de otro documento.

Ejemplo:

<blockquote cite="https://www.ejemplo.com">
    <p>Cita larga...</p>
</blockquote>

<body>

Descripción: La etiqueta <body> contiene el contenido visible del documento HTML.

Ejemplo:

<body>
    <h1>Encabezado</h1>
    <p>Texto del cuerpo</p>
</body>

<br>

Descripción: La etiqueta <br> inserta un salto de línea, útil para formatear texto.

Ejemplo:

Texto antes del salto<br>Texto después del salto

<button>

Descripción: La etiqueta <button> define un botón interactivo.

Ejemplo:

<button type="button">Clic aquí</button>

<canvas>

Descripción: La etiqueta <canvas> se utiliza para dibujar gráficos mediante JavaScript.

Ejemplo:

<canvas id="miCanvas" width="200" height="100"></canvas>

<caption>

Descripción: La etiqueta <caption> proporciona un título para un elemento <table>.

Ejemplo:

<table>
    <caption>Título de la tabla</caption>
    <tr><th>Encabezado</th></tr>
</table>

<cite>

Descripción: La etiqueta <cite> se usa para citar el título de una obra, como un libro o una película.

Ejemplo:

<cite>El Quijote</cite>

<code>

Descripción: La etiqueta <code> define un fragmento de código en línea.

Ejemplo:

<code>console.log("Hola Mundo");</code>

<col>

Descripción: La etiqueta <col> define una columna en una tabla y se usa dentro de un elemento <colgroup>.

Ejemplo:

<colgroup>
    <col span="2" style="background-color: #f4f4f4">
    <col style="background-color: #fff">
</colgroup>

<colgroup>

Descripción: La etiqueta <colgroup> define un grupo de columnas en una tabla, aplicando estilos a todas las columnas del grupo.

Ejemplo:

<colgroup>
    <col style="background-color: #f4f4f4">
    <col style="background-color: #e2e2e2">
</colgroup>

<data>

Descripción: La etiqueta <data> asocia un valor con un contenido. Es útil para máquinas y procesos automáticos.

Ejemplo:

<data value="2024-08-30">30 de agosto de 2024</data>

<datalist>

Descripción: La etiqueta <datalist> contiene una lista de opciones predefinidas para un campo <input>.

Ejemplo:

<input list="browsers">
<datalist id="browsers">
    <option value="Chrome">
    <option value="Firefox">
    <option value="Safari">
</datalist>

<dd>

Descripción: La etiqueta <dd> define una descripción o valor en una lista de descripción.

Ejemplo:

<dl>
    <dt>Etiqueta</dt>
    <dd>Descripción de la etiqueta</dd>
</dl>

<del>

Descripción: La etiqueta <del> muestra texto que ha sido eliminado o modificado, generalmente con una línea a través.

Ejemplo:

<del>Texto eliminado</del>

<details>

Descripción: La etiqueta <details> define una sección de detalles que el usuario puede ver u ocultar.

Ejemplo:

<details>
    <summary>Más detalles</summary>
    <p>Contenido adicional...</p>
</details>

<dfn>

Descripción: La etiqueta <dfn> define una definición de un término dentro de un documento.

Ejemplo:

<dfn>HTML</dfn> es un lenguaje de marcado.

<dialog>

Descripción: La etiqueta <dialog> define un cuadro de diálogo o ventana emergente.

Ejemplo:

<dialog open>
    <p>Este es un cuadro de diálogo</p>
    <button>Cerrar</button>
</dialog>

<div>

Descripción: La etiqueta <div> define una sección genérica en un documento. Se utiliza para agrupar contenido.

Ejemplo:

<div>
    <h2>Título</h2>
    <p>Contenido dentro de un div</p>
</div>

<dl>

Descripción: La etiqueta <dl> define una lista de descripción, que consta de términos y descripciones.

Ejemplo:

<dl>
    <dt>Término</dt>
    <dd>Descripción del término</dd>
</dl>

<dt>

Descripción: La etiqueta <dt> define un término en una lista de descripción.

Ejemplo:

<dl>
    <dt>HTML</dt>
    <dd>Lenguaje de marcado</dd>
</dl>

<em>

Descripción: La etiqueta <em> define texto enfatizado. Usualmente se muestra en cursiva.

Ejemplo:

<em>Texto enfatizado</em>

<embed>

Descripción: La etiqueta <embed> se usa para incrustar contenido externo, como videos o archivos multimedia.

Ejemplo:

<embed src="video.mp4" type="video/mp4">

<fieldset>

Descripción: La etiqueta <fieldset> agrupa elementos en un formulario y proporciona una estructura visual.

Ejemplo:

<fieldset>
    <legend>Información Personal</legend>
    <input type="text">
</fieldset>

<figcaption>

Descripción: La etiqueta <figcaption> proporciona un título o leyenda para un elemento <figure>.

Ejemplo:

<figure>
    <img src="imagen.jpg" alt="Descripción">
    <figcaption>Descripción de la imagen</figcaption>
</figure>

<figure>

Descripción: La etiqueta <figure> representa contenido independiente, como una imagen o un gráfico, junto con su leyenda.

Ejemplo:

<figure>
    <img src="imagen.jpg" alt="Descripción">
    <figcaption>Descripción de la imagen</figcaption>
</figure>

<footer>

Descripción: La etiqueta <footer> define el pie de una sección o documento, comúnmente contiene información de contacto o derechos de autor.

Ejemplo:

<footer>
    <p>© 2024 Ejemplo Inc.</p>
</footer>

<form>

Descripción: La etiqueta <form> define un formulario de entrada de datos.

Ejemplo:

<form action="/submit" method="post">
    <input type="text" name="nombre">
    <input type="submit" value="Enviar">
</form>

<h1>

Descripción: La etiqueta <h1> define el encabezado principal del documento, siendo el nivel más alto.

Ejemplo:

<h1>Título Principal</h1>

<h2>

Descripción: La etiqueta <h2> define un subtítulo o encabezado de segundo nivel.

Ejemplo:

<h2>Subtítulo</h2>

<h3>

Descripción: La etiqueta <h3> define un encabezado de tercer nivel.

Ejemplo:

<h3>Encabezado de Tercer Nivel</h3>

<h4>

Descripción: La etiqueta <h4> define un encabezado de cuarto nivel.

Ejemplo:

<h4>Encabezado de Cuarto Nivel</h4>

<h5>

Descripción: La etiqueta <h5> define un encabezado de quinto nivel.

Ejemplo:

<h5>Encabezado de Quinto Nivel</h5>

<h6>

Descripción: La etiqueta <h6> define un encabezado de sexto nivel, el más bajo.

Ejemplo:

<h6>Encabezado de Sexto Nivel</h6>

<head>

Descripción: La etiqueta <head> contiene metadatos sobre el documento, como el título, los enlaces a hojas de estilo y scripts.

Ejemplo:

<head>
    <title>Título de la Página</title>
    <link rel="stylesheet" href="estilos.css">
</head>

<header>

Descripción: La etiqueta <header> define el encabezado de una sección o documento. Suele contener logotipos, menús de navegación, etc.

Ejemplo:

<header>
    <h1>Encabezado del Documento</h1>
</header>

<hgroup>

Descripción: La etiqueta <hgroup> agrupa un conjunto de encabezados.

Ejemplo:

<hgroup>
    <h1>Título Principal</h1>
    <h2>Subtítulo</h2>
</hgroup>

<hr>

Descripción: La etiqueta <hr> inserta una línea horizontal para separar contenido.

Ejemplo:

<hr>

<html>

Descripción: La etiqueta <html> es el contenedor principal del documento HTML, que engloba todo el contenido de la página.

Ejemplo:

<html>
    <head></head>
    <body></body>
</html>

<i>

Descripción: La etiqueta <i> define texto en cursiva, pero sin un significado semántico especial.

Ejemplo:

<i>Texto en cursiva</i>

<iframe>

Descripción: La etiqueta <iframe> define un marco en línea que permite incrustar otro documento HTML dentro de la página.

Ejemplo:

<iframe src="https://www.ejemplo.com" width="600" height="400"></iframe>

<img>

Descripción: La etiqueta <img> incrusta una imagen en el documento.

Ejemplo:

<img src="imagen.jpg" alt="Descripción">

<input>

Descripción: La etiqueta <input> define un campo de entrada de datos.

Ejemplo:

<input type="text" name="nombre">

<ins>

Descripción: La etiqueta <ins> muestra texto que ha sido insertado en el documento, generalmente con un subrayado.

Ejemplo:

<ins>Texto insertado</ins>

<kbd>

Descripción: La etiqueta <kbd> define texto que representa la entrada del teclado.

Ejemplo:

<kbd>Ctrl + C</kbd>

<label>

Descripción: La etiqueta <label> define una etiqueta para un elemento de formulario.

Ejemplo:

<label for="nombre">Nombre:</label>
<input type="text" id="nombre" name="nombre">

<legend>

Descripción: La etiqueta <legend> define un título o leyenda para un elemento <fieldset>.

Ejemplo:

<fieldset>
    <legend>Información Personal</legend>
    <input type="text">
</fieldset>

<li>

Descripción: La etiqueta <li> define un elemento de una lista ordenada o desordenada.

Ejemplo:

<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ul>

<main>

Descripción: La etiqueta <main> define el contenido principal del documento, excluyendo encabezados, pie de página y barras laterales.

Ejemplo:

<main>
    <h1>Contenido Principal</h1>
</main>

<map>

Descripción: La etiqueta <map> define un mapa de imagen, proporcionando áreas clicables dentro de una imagen.

Ejemplo:

<map name="mapa" id="mapa">
    <area shape="rect" coords="34,44,270,350" href="pagina.html" alt="Área clickeable">
</map>

<mark>

Descripción: La etiqueta <mark> resalta texto como si fuera una marca, generalmente con un fondo amarillo.

Ejemplo:

<mark>Texto resaltado</mark>

<meta>

Descripción: La etiqueta <meta> define metadatos sobre el documento, como la codificación de caracteres y el autor.

Ejemplo:

<meta charset="UTF-8">

<meter>

Descripción: La etiqueta <meter> define una medida escalar dentro de un rango conocido, como una barra de progreso.

Ejemplo:

<meter value="3" min="0" max="10">3 de 10</meter>

<nav>

Descripción: La etiqueta <nav> define un bloque de navegación con enlaces a otras páginas o secciones del mismo documento.

Ejemplo:

<nav>
    <a href="#home">Inicio</a>
    <a href="#about">Acerca de</a>
</nav>

<noscript>

Descripción: La etiqueta <noscript> define un contenido alternativo para los usuarios cuyo navegador no soporta scripts.

Ejemplo:

<noscript>Tu navegador no soporta JavaScript.</noscript>

<object>

Descripción: La etiqueta <object> define un objeto incrustado, como un archivo multimedia o un documento.

Ejemplo:

<object data="archivo.pdf" type="application/pdf"></object>

<ol>

Descripción: La etiqueta <ol> define una lista ordenada.

Ejemplo:

<ol>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ol>

<optgroup>

Descripción: La etiqueta <optgroup> agrupa opciones dentro de una lista desplegable.

Ejemplo:

<select>
    <optgroup label="Frutas">
        <option>Manzana</option>
        <option>Naranja</option>
    </optgroup>
    <optgroup label="Verduras">
        <option>Zanahoria</option>
        <option>Brócoli</option>
    </optgroup>
</select>

<option>

Descripción: La etiqueta <option> define una opción dentro de un menú desplegable.

Ejemplo:

<select>
    <option value="1">Opción 1</option>
    <option value="2">Opción 2</option>
</select>

<output>

Descripción: La etiqueta <output> define el resultado de una acción del usuario, como el resultado de un cálculo.

Ejemplo:

<output for="form1" name="resultado">Resultado</output>

<p>

Descripción: La etiqueta <p> define un párrafo de texto.

Ejemplo:

<p>Este es un párrafo.</p>

<param>

Descripción: La etiqueta <param> define parámetros para un objeto incrustado.

Ejemplo:

<object data="video.mp4">
    <param name="autoplay" value="true">
</object>

<picture>

Descripción: La etiqueta <picture> proporciona varias opciones de imagen para adaptarse a diferentes resoluciones y condiciones de visualización.

Ejemplo:

<picture>
    <source srcset="imagen-800.jpg" media="(min-width: 800px)">
    <source srcset="imagen-400.jpg" media="(max-width: 799px)">
    <img src="imagen-400.jpg" alt="Imagen adaptable">
</picture>

<pre>

Descripción: La etiqueta <pre> define texto preformateado que conserva los espacios y saltos de línea.

Ejemplo:

<pre>
    Texto
    con
    formato
</pre>

<progress>

Descripción: La etiqueta <progress> muestra el progreso de una tarea.

Ejemplo:

<progress value="50" max="100">50%</progress>

<q>

Descripción: La etiqueta <q> define una cita corta dentro del texto.

Ejemplo:

<q>Esta es una cita corta.</q>

<rp>

Descripción: La etiqueta <rp> define el texto que se muestra en navegadores que no soportan elementos <ruby>.

Ejemplo:

<ruby>
    <rp>(</rp>
    <rt>Texto en ruby</rt>
    <rp>)</rp>
</ruby>

<rt>

Descripción: La etiqueta <rt> define el texto anotado en un elemento <ruby>.

Ejemplo:

<ruby>
    <rt>Texto anotado</rt>
    <ruby>Texto principal</ruby>
</ruby>

<s>

Descripción: La etiqueta <s> define texto que debe ser mostrado como tachado.

Ejemplo:

<s>Texto tachado</s>

<samp>

Descripción: La etiqueta <samp> define una muestra de texto de salida de un programa.

Ejemplo:

<samp>Texto de salida</samp>

<script>

Descripción: La etiqueta <script> define un bloque de código JavaScript.

Ejemplo:

<script>
    alert('Hola, mundo!');
</script>

<section>

Descripción: La etiqueta <section> define una sección del documento.

Ejemplo:

<section>
    <h2>Título de Sección</h2>
    <p>Contenido de la sección.</p>
</section>

<select>

Descripción: La etiqueta <select> define un menú desplegable.

Ejemplo:

<select>
    <option>Opción 1</option>
    <option>Opción 2</option>
</select>

<small>

Descripción: La etiqueta <small> define texto de menor tamaño.

Ejemplo:

<small>Texto pequeño</small>

<span>

Descripción: La etiqueta <span> define una sección en línea para aplicar estilos o scripts.

Ejemplo:

<span style="color:red">Texto en rojo</span>

<strong>

Descripción: La etiqueta <strong> define texto importante, generalmente en negrita.

Ejemplo:

<strong>Texto importante</strong>

<style>

Descripción: La etiqueta <style> define hojas de estilo CSS dentro del documento HTML.

Ejemplo:

<style>
    body {
        font-family: Arial, sans-serif;
    }
</style>

<sub>

Descripción: La etiqueta <sub> define texto en subíndice.

Ejemplo:

<sub>Texto en subíndice</sub>

<summary>

Descripción: La etiqueta <summary> define un resumen o título para un elemento <details>.

Ejemplo:

<details>
    <summary>Más información</summary>
    <p>Detalles adicionales</p>
</details>

<sup>

Descripción: La etiqueta <sup> define texto en superíndice.

Ejemplo:

<sup>Texto en superíndice</sup>

<table>

Descripción: La etiqueta <table> define una tabla en el documento.

Ejemplo:

<table>
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
    <tr>
        <td>Celda 1</td>
        <td>Celda 2</td>
    </tr>
</table>

Descripción: La etiqueta <tbody> agrupa el cuerpo de una tabla, separándolo del encabezado y el pie.

Ejemplo:

<table>
    <thead>
        <tr><th>Encabezado</th></tr>
    </thead>
    <tbody>
        <tr><td>Celda 1</td></tr>
    </tbody>
</table>

<td>

Descripción: La etiqueta <td> define una celda en una tabla.

Ejemplo:

<table>
    <tr>
        <td>Celda 1</td>
        <td>Celda 2</td>
    </tr>
</table>

<textarea>

Descripción: La etiqueta <textarea> define un área de texto multilínea.

Ejemplo:

<textarea rows="4" cols="50">Texto</textarea>

<tfoot>

Descripción: La etiqueta <tfoot> define el pie de una tabla.

Ejemplo:

<table>
    <tfoot>
        <tr><td>Pie de tabla</td></tr>
    </tfoot>
</table>

<table>
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
</table>

<thead>

Descripción: La etiqueta <thead> agrupa el encabezado de una tabla, separándolo del cuerpo y el pie.

Ejemplo:

<table>
    <thead>
        <tr><th>Encabezado</th></tr>
    </thead>
    <tbody>
        <tr><td>Celda</td></tr>
    </tbody>
</table>

<time>

Descripción: La etiqueta <time> define una fecha o una hora.

Ejemplo:

<time datetime="2024-08-30T12:00">30 de agosto de 2024</time>

<title>

Descripción: La etiqueta <title> define el título del documento, que aparece en la barra de título del navegador o en las pestañas.

Ejemplo:

<title>Mi Página Web</title>

<tr>

Descripción: La etiqueta <tr> define una fila en una tabla.

Ejemplo:

<table>
    <tr>
        <td>Celda 1</td>
        <td>Celda 2</td>
    </tr>
</table>

<u>

Descripción: La etiqueta <u> define texto subrayado.

Ejemplo:

<u>Texto subrayado</u>

Descripción: La etiqueta <ul> define una lista desordenada.

Ejemplo:

<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ul>

<var>

Descripción: La etiqueta <var> define una variable en un contexto matemático o de programación.

Ejemplo:

<var>x</var> = 10

<video>

Descripción: La etiqueta <video> define un video incrustado en el documento.
